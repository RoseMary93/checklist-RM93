# 📌 RM93 待辦清單

一個功能完整的桌面待辦事項管理應用程式，採用 PyQt6 開發。提供直覺的使用介面、工作任務追蹤、歷史紀錄查詢等功能，幫助使用者有效管理日常工作清單。

## ✨ 功能特點

- **任務管理**
  - ✏️ 新增、編輯、刪除任務
  - ☑️ 勾選標記任務完成/未完成
  - ↑↓ 自由拖拽或按鈕調整任務優先順序
  
- **視覺分類**
  - 📝 待辦事項區：顯示所有未完成的任務
  - ✅ 本日完成區：自動分類今日完成的任務
  - 🎨 溫暖的黃色配色，舒適長時間閱讀
  
- **歷史紀錄**
  - 📊 表格式查看所有任務歷史
  - 🕐 記錄任務的建立時間、完成時間
  - 📈 追蹤任務完成狀態
  
- **便捷功能**
  - 📌 釘選功能：保持應用視窗始終可見
  - 💾 自動持久化：所有資料自動儲存至 SQLite 資料庫
  - ⌨️ 快速輸入：按 Enter 快速新增任務

## 🛠️ 技術棧

- **框架**: PyQt6
- **資料庫**: SQLite3
- **語言**: Python 3.8+
- **UI 特性**: 自訂樣式表、響應式佈局、無邊框視窗

## 📋 系統要求

- Windows / macOS / Linux
- Python 3.8 或更高版本
- PyQt6

```bash

```

## 📖 使用指南

### 基本操作

| 操作 | 說明 |
|------|------|
| **新增任務** | 在輸入框輸入任務名稱，按 `Enter` 或點擊新增 |
| **完成任務** | 點擊任務左側的勾選框 ☑️ 標記完成 |
| **編輯任務** | 點擊任務右側的編輯按鈕 ✎ 修改名稱 |
| **刪除任務** | 點擊任務右側的刪除按鈕 ✕ 移除任務 |
| **調整優先順序** | 點擊 ↑ ↓ 按鈕上下移動任務，或直接拖拽 |

### 進階功能

- **查看歷史紀錄**: 點擊左下方「歷史紀錄」按鈕開啟歷史紀錄視窗
- **釘選視窗**: 點擊「📌 釘選」按鈕，應用將保持在其他視窗上方
- **關閉應用**: 點擊「關閉便籤」按鈕或點擊視窗關閉按鈕

### 資料持久化

- 所有任務資料自動儲存至同目錄下的 `todo_history.db` SQLite 資料庫
- 應用關閉後重新啟動，所有任務會自動恢復
- 本日完成的任務會在隔天自動歸檔至歷史紀錄

## 💡 核心代碼架構

### 主要類別

- **HistoryWindow**: 歷史紀錄視窗，表格式展示所有任務
- **TodoItem**: 單個任務項目元件，管理任務的各項交互
- **DesktopTodo**: 主應用程式類別，管理整體 UI 和資料庫操作

### 資料庫結構

```sql
CREATE TABLE tasks (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    task TEXT,           -- 任務描述
    created TEXT,        -- 建立時間
    finished TEXT,       -- 完成時間
    is_done INTEGER,     -- 完成狀態 (0: 未完成, 1: 完成)
    position INTEGER     -- 任務排序位置
)
```

## 🎨 自訂外觀

應用主要使用 QSS（Qt 樣式表）自訂外觀。可在 `init_ui()` 方法中修改以下內容：

- **背景色**: `#FFFACD` (淡黃色)
- **字型**: 微軟正黑體，13px
- **按鈕顏色**: `#F0E68C` (金色)
- **邊框色**: `#E6DB74`

## 🔧 故障排除

| 問題 | 解決方案 |
|------|----------|
| 找不到 `checklist_logo.png` | 應用會自動跳過缺失的圖標，不影響功能 |
| PyQt6 導入錯誤 | 確認虛擬環境已啟動，重新執行 `pip install PyQt6` |
| 資料庫鎖定錯誤 | 確保只有一個應用實例執行，或刪除 `.db` 文件重新開始 |
